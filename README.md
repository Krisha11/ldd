# ldd

Сборка (в корне)

```
g++ -o main main.cpp worker/ldd_worker.cpp -I worker
```


Запуск

```
./main <file>
```

Тестирование (из корня)

```
python test.py
```


Версия утилиты `ldd`, показывающая все зависимости указанного бинарного модуля, в том числе и транзитивные (зависимости зависимостей).
Также утилита способна установить "ожидаемую" схему импорта: какой из импортируемых символов в какой библиотеке ожидается найти.

Список путей, по которым ищутся библиотеки, формируется из  LD_LIBRARY_PATH + /lib + /usr/lib
